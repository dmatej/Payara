list-log-attributes

lists all logging attributes defined for a specified target in a domain

Synopsis

asadmin [asadmin-options] list-log-attributes [--help]
[target]

Description

The list-log-attributes subcommand lists all logging attributes currently defined for the specified \{product---name} domain or target within a domain. The values listed correspond to the values in the logging.properties file for the domain.

This subcommand is supported in remote mode only.

Options

asadmin-options

Options for the asadmin utility. For information about these options, see the asadmin(1M) help page.

--help
-?

Displays the help text for the subcommand.

Operands

target

Valid values are:

  • server - The default server instance. This is the default value.

  • configuration_name - The name of a specific configuration.

  • cluster_name - The name of a target cluster.

  • instance_name - The name of a target server instance.
